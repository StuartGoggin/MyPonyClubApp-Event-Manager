## How to Review and Approve Events

### Finding Events to Review

1. **Look for Pending Events**
   - Events with "Proposed" status need your attention
   - They appear in the **Pending Approvals** section
   - Badge shows number waiting for review

2. **Click on an Event**
   - View full details in the modal/popup
   - Review all information carefully

### What to Check ‚úÖ

Before approving, verify:

- [ ] **Event Type** - Is it correctly classified? (Rally, Camp, Competition, etc.)
- [ ] **Date & Time** - Does it conflict with other events?
- [ ] **Public Holiday** - Is it on a holiday? (System shows warnings)
- [ ] **Location** - Is the venue appropriate and available?
- [ ] **Description** - Is there enough detail?
- [ ] **Club** - Is this the correct club?
- [ ] **Zone** - Is it in your zone?

### To APPROVE an Event ‚úÖ

1. Click the **"Approve"** button (green checkmark)
2. Optionally add approval notes:
   - "Great planning!"
   - "Please confirm venue booking"
   - "Ensure insurance is in place"
3. Click **"Confirm Approval"**

**What Happens:**
- Event status changes to "Approved"
- Club manager receives email notification
- Event appears on zone calendar
- Club can now upload schedule and proceed with planning

### To REJECT an Event ‚ùå

1. Click the **"Reject"** button (red X)
2. **You MUST provide feedback** explaining:
   - What needs to be changed
   - Why you're rejecting it
   - How they can fix it

**Example Feedback:**
```
This event conflicts with Bayside Pony Club's rally on the same date. 
Please reschedule to the following weekend (Dec 22) and resubmit. 
Also, please provide more details about the competition format.
```

3. Click **"Confirm Rejection"**

**What Happens:**
- Event status changes to "Rejected"
- Club manager receives email with your feedback
- Club can edit and resubmit
- You'll see it again when they resubmit

### Adding Comments (Without Approve/Reject)

**Sometimes you need more information:**

1. Click the **Comment** icon
2. Ask questions or request clarification:
   - "Can you provide the judge's name?"
   - "Will this be open to other zones?"
   - "Please confirm catering arrangements"
3. Club manager receives notification
4. They can respond in the comments section
5. You can still approve/reject later

## Common Scenarios

### Scenario 1: Date Conflict

**Problem:** Two clubs want the same date

**Solution:**
1. Check which event was submitted first
2. Consider event importance and tradition
3. Contact both clubs
4. Work out an arrangement
5. Approve one, provide feedback to other suggesting alternate date

### Scenario 2: Public Holiday Event

**Problem:** Event scheduled on public holiday

**System Shows:** ‚ö†Ô∏è Warning banner

**Your Options:**
- **Approve** if it's intentional (some events run on holidays)
- **Reject** if club didn't realize
- **Comment** to confirm their intention

### Scenario 3: Incomplete Information

**Problem:** Event missing details

**Solution:**
1. Click **Comment**
2. List what's missing:
   - "Please add event start/end times"
   - "Please specify if open to other clubs"
3. **Don't reject** - give them a chance to add info
4. Approve once information is complete

### Scenario 4: Annual Event

**Problem:** Same event as last year

**Solution:**
- Review as normal (don't auto-approve)
- Check dates work for current year
- Verify location is still suitable
- Approve if all looks good

## Tips for Success

### ‚úÖ Best Practices

1. **Check Daily**
   - Review pending approvals at least once a day
   - Don't leave clubs waiting

2. **Be Thorough**
   - Check all details before approving
   - Better to ask questions than approve incorrectly

3. **Communicate Clearly**
   - When rejecting, be specific about what needs fixing
   - Use friendly, helpful language
   - Remember you're supporting clubs, not blocking them

4. **Stay Organized**
   - Keep track of annual events
   - Note venue availability patterns
   - Watch for seasonal conflicts

5. **Be Fair**
   - First-come-first-served for date conflicts
   - Consider all clubs equally
   - Document your decisions

### üö´ Common Mistakes to Avoid

1. **Don't approve without reviewing**
   - Always check for conflicts
   - Verify all details

2. **Don't reject without feedback**
   - Clubs need to know what to fix
   - Generic rejections aren't helpful

3. **Don't delay unnecessarily**
   - Clubs are planning; time matters
   - If you need info, ask quickly

4. **Don't ignore public holiday warnings**
   - System flags these for a reason
   - Confirm with club if intentional
